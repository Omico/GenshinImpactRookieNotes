# 礼包商城

推荐购买空月祝福，相对于其他优惠力度最大。有能力的旅行者可以选择购买。

除空月祝福以外的其他物品，懒人重氪旅行者请根据自己实际需求购买。真的没有什么必要。

# 星辉兑换

请根据自身需求兑换纠缠之缘、相遇之缘、角色、武器。

其他突破用素材不推荐兑换。

# 星尘兑换

请把打折的纠缠之缘、相遇之缘优先级设为最高。

其次推荐兑换打折的摩拉。

# 原石购买

用原石购买相遇之缘属于浪费行为，请优先购买纠缠之缘。

具体的说明请移步抽卡建议。
