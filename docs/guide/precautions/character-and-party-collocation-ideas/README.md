# 角色与阵容的配装思路

一个角色在战斗中发挥效果被很多因素所影响。
本文不拘泥于 Wiki 上所谓的推荐配装。
因为每个账号当前拥有的圣遗物都是不同的，根据自身条件灵活变通才是旅行者应该去做的。
笔者在这期间发现身边的萌新旅行者们都过于相信所谓的推荐配装。
~~结果笔者上号一看，直接把未强化的圣遗物替换已强化的圣遗物，角色伤害不变。~~

由此这篇文章应运而生。

::: tip 专家建议

**首先旅行者们要知道角色在战斗中发挥效果是根据角色所持有的武器、圣遗物，以及角色的天赋命之座等级。**

**千万不要强化任何四星以下的圣遗物或武器。**

**武器的星级越高，武器的上限也就越高，因此从一开始旅行者就应该选择使用至少为四星的武器。但需注意的是，在三星武器中有一个名为【讨龙英杰谭】的法器，其猴戏效果拉满。因此，请收集并升至精炼 5 备用。**

> 主动切换角色时，新登场的角色攻击力提升 24%/30%/36%/42%/48%，持续 10 秒。该效果每 20 秒只能触发一次。

**圣遗物与武器同理，星级越高，上限也就越高。但强化圣遗物属于赌狗行为，因此一定不要看到圣遗物就强化，要根据副属性去做判断。**

**被强化的圣遗物可作为高级狗粮，但只能给予其 80% 的经验值，请谨慎判断其得失。**

**一定要记住不论如何，元素反应永远的神！对不起，元素反应就是可以为所欲为！**

**如果想要确保自己的游戏体验，切记不要一次培养多个角色~~海王行为~~，资源是完全不够的。**

**推荐流程为，前期先培养一位主 C 和主奶，然后培养副 C 辅助或副奶，中期培养第二队，后期则可以根据自己喜好去培养~~挥霍~~角色。**

不管什么时期角色培养都应该分为两个阶段：

- 基础属性培养
  - 角色
    - 等级
    - 天赋等级
  - 武器
    - 等级
    - 精炼等级
- 圣遗物配装

## 基础属性培养思路

任何时期，都应该先从角色的等级、天赋等级、武器等级开始进行培养。

**以上提到的提升都是实打实的，而圣遗物纯属赌狗行为，都是运气。请不要没有欧皇的命，得了赌鬼的病。**

**首先旅行者们需要知道，在一个队伍中主要分为主 C、副 C、辅助、主奶、副奶，且同一个角色在队伍中可能拥有多种职责。**

本节将根据职责划分进行具体的指导。

### 主 C

主 C 即占场 C，顾名思义就是在战斗中进行持续输出的。

推荐培养要求：

- 角色/武器等级为当前限制内最高或至少 80 级
- 角色所有天赋至少为 8 级（有命之座加成的也行，但还是建议尽量往上拉）

### 副 C

副 C 包含功能 C 和脱手 C 两种。

推荐培养要求请参考辅助。

#### 功能 C

功能 C 是一种比较特殊的定位，能在一定时间内打出高额伤害的角色都可以被定义为功能 C

例如：行秋，胡桃，魈（未满命），公子（未满命），钟离，诺埃尔，等等。

**这里需要注意，对角色是否为功能 C 的定义与角色所在的队伍以及角色配装有很大关系。请旅行者自行思考，而不是照搬。**

由上定义，因为不能持续进行输出，在队伍仅包含一个功能 C 而没有占场 C 时，往往会对队伍产生负面效果。

于是演变出双功能 C 交替输出占场流，请看[角色定位示例](#角色定位示例)。

#### 脱手 C

脱手 C 顾名思义，角色处于后台时也能对战场实行高额伤害。

例如：莫娜，香菱等。

### 辅助

对占场 C 提供支援的都可以被称为辅助。副 C 和奶位都被包含在辅助，但与其不同的是，辅助还包括给予护盾，角色技能的特殊效果。

例如：迪奥娜，诺埃尔，辛焱，钟离等。

推荐培养要求：

- 角色/武器等级尽量提升
- 与辅助定位相关的角色天赋至少为 8 级（有命之座加成的也行，但还是建议尽量往上拉）

### 主奶

持续为队伍回复大量生命值。

例如：芭芭拉，七七等。

推荐培养要求请参考辅助。

### 副奶

由于主奶奶量不足而产生辅助位。同时，角色自身特性能够回复其他角色生命的也被归类于此。

例如：诺埃尔，钟离（满命），行秋等。

推荐培养要求请参考辅助。

### 角色定位示例

这里举个例子，下面列举出笔者日常跑图比较常用阵容，在此只讲思路，不对具体数值进行说明：

- 阵容一（打 Boss，例如：无相之岩、北风狼、公子等）

  - 胡桃（功能 C，伪主 C）
  - 可莉（功能 C）练度不够；且有胡桃，不需要持续占场
  - 钟离（功能 C，辅助）护盾工具人，大招工具人，未满命不能回血
  - 诺埃尔（功能 C，辅助，伪主奶）因为盾多，且胡桃不需要回复，所以诺埃尔的奶量已经足够了

- 阵容二（日常跑图，刷取日常任务或地脉，风本这个阵容单刷只需 30s）

  - 魈（功能 C，伪主 C）我是 0 命，因此需要温蒂的技能以及自身两段元素战技才能有机会充满大招
  - 温蒂（辅助，脱手 C）练度比较高，伤害比较高，因此也被划归为脱手 C
  - 钟离（功能 C，辅助）同阵容一
  - 诺埃尔（功能 C，辅助，伪主奶）同阵容一

以上阵容是在笔者练度的情况下才适用的，这里只是作为一个启发性举例，当然日常针对某些特殊的场景，例如：风魔龙，爆炎树等，会对上述阵容一有所调整。比如，打风魔龙会把可莉替换为甘雨（破盾，大招与胡桃配合，轻松秒杀）；打爆炎树会把可莉换成行秋（武器为满精祭礼剑，两段元素战技后开大配合胡桃，轻松秒杀）。

## 圣遗物配装思路

本文将把旅行者分为以下五个阶段，旅行者们可以根据自身情况选择阅读。

### 前期（三星以下圣遗物为主）

在这一时期，旅行者们应该把资源重点给予主角（空/萤）身上（氪佬请略过）。
把主角升至 40 级后基本就可以停止了。原因有以下几点：

- 风系主角 = 反应工具人
- 岩系主角 = 机关工具人
- 后面抽卡后会有更好的角色入队

因此，完全没必要一直拘泥于主角。

还有一个需要特别注意的点是萤的重击伤害加成比空要高，具体数值请在查看 [Wiki](https://bbs.mihoyo.com/ys/obc/content/53/detail?bbs_presentation_style=no_header)。

~~妹妹是暴力女，母猩猩；哥哥是受实锤了~~

**这一时期玩家因把提升的重心侧重于角色的天赋和武器上的升级上。**

**完全不需要升级圣遗物。注意是完全！**

**暴击率和暴击伤害的词缀在这个时期基本上没多少作用，加成太少了。主词缀无脑堆攻击力完事了。**

圣遗物的选择请根据当前已有圣遗物的套装效果而定，这里举个例子。

例如风主，作为反应工具人，应该要有那么一点点风元素伤害加成。那么空之杯应该尽量选择风元素伤害的，其次按顺序依次是攻击力，元素精通，元素充能效率，物理伤害。其余 4 个可以根据自身喜好随意搭配。

又比如岩主，前期被圣遗物限制，战斗中作用比风主低，因此空之杯应该尽量选择岩元素伤害的，其次按顺序依次是攻击力，物理伤害，元素充能效率。

武器方面，铁匠处可以打造两种武器：

- 试作斩岩

  - 副词缀为物理伤害
  - 适合岩主，风主也能用
  - 原因：风主和岩主的普攻伤害都是白字，也就是物理伤害

- 铁蜂刺

  - 副词缀为元素精通
  - 适合风主，岩主也能用，但只推荐手残玩家使用
  - 原因：元素反应的伤害与元素精通有关，而岩系角色的元素精通仅仅只是增加护盾伤害吸收量，怪都碰不到你，自然不需要护盾。

### 中期（以四星圣遗物为主）

在出现紫色的四星圣遗物后，旅行者需要注意，你已经向着新的阶段迈进了。

这时相信旅行者已经有了新的角色，因此主角也可以下场了。

这个时期建议逐步强化四星圣遗物中主词缀匹配的圣遗物。

### 中后期（出现五星圣遗物）

这是一个过渡阶段，对应冒险等级为 50。

武器 50 级后的副属性的暴击率，【战狂的蔷薇】二件套效果和主词缀为暴击率的理之冠之间搭配一下，暴击率能超过 50% 后爆伤就开始变得重要起来。

这里的主要装配思路是武器和圣遗物之间，先使暴击率达到 50%然后开始堆爆伤，爆伤是越高越好。

作为主 C，尽量使其暴击率爆伤分别在 50% 和 150% 左右，堆不到 150% 也没关系，尽量就行。而且之后五星圣遗物不在稀有后，还是会替换的。因此四星圣遗物能用就行，不要刻意刷，不过前提是主词缀一定要对，不能因为无脑堆暴击率爆伤而牺牲其他属性。

例如凝光主 C，首先旅行者要知道，凝光作为岩系角色，元素精通是增加护盾伤害吸收的。那么作为主 C 就不应该过分追求这个，因此套装效果是攻击力、暴击率、元素充能、普通攻击伤害、元素战技伤害的圣遗物都可以根据自身喜好进行搭配，因为牵扯到具体攻击力和暴击爆伤的数值，这里只能旅行者自行判断，去找到最优解。圣遗物副词缀同理

主词缀应按照以下思路。

- 生之花 生命百分比（固定）
- 死之羽 攻击力（固定）
- 时之沙 攻击力百分比
- 空之杯 岩元素伤害加成（其次为攻击力）
- 理之冠 暴击率/暴击伤害（取决于武器副属性词缀，圣遗物套装效果）

Wiki 上所谓的推荐装配在这一时期并不是绝对的，不必过分纠结。因根据背包内圣遗物情况，动态调整。

出现五星圣遗物后不要激动，或许词缀不好还不如你当前的 4 星圣遗物。

### 后期（以五星圣遗物为主）

这一时期可以为角色升满一套四星也就是紫色的圣遗物，然后等到可以刷四级秘境的时候，使用之前积攒浓缩树脂的全力去刷圣遗物。此时你可以根据 Wiki 或 B 站 UP 主的推荐去刷圣遗物。

这个时期圣遗物的刷取建议是，先寻找主词缀符合条件的五星圣遗物，替换身上的四星圣遗物。当然你也可以一步到位，刷出合适的主/副词缀都对的圣遗物。当角色身上的圣遗物已经全部替换为 5 星后，你该着重于副属性的提升，使角色成为一个正常人。

### 大后期（老年旅行者）

对应冒险等级为 55+。

这一时期就是笔者现在的时期，每天上线清理体力，刷取各种副本，获取有价值的圣遗物胚子，然后随缘强化。尽量往上限靠拢。
